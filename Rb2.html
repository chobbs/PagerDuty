Check databases disk-space/

SSH into target server(s) and execute the  df command following command:

df -k

Smaple Output:

  • “Filesystem” - The name of the filesystem.
  • “1K-blocks” - The size of the filesystem in 1K blocks.
  • “Used” - The used space in 1K blocks.
  • “Available” - The available space in 1K blocks.
  • “Use%” - The percentage of used space.
  • “Mounted on” the directory on which the filesystem is mounted.

To display information only for a specific file system, pass its name or the mount point to the df command.

Restart MYSQL om Linux/

SSH into target server(s) and execute the  service command following command:

 

You use the following command to restart the MySQL server On Linux:

 
service mysql restart 
Code language: SQL (Structured Query Language) (sql)

If the name is MySQL service is mysqld not mysql, you need to change the service name in the command as shown in the following command:

 
service mysqld restart 
Code language: SQL (Structured Query Language) (sql)

Or you can use the init.d to start the MySQL service:

 
/etc/init.d/mysqld restart 
Code language: SQL (Structured Query Language) (sql)

In this tutorial, you have learned how to restart the MySQL server on Windows and Linux.
